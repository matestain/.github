# MATESTAIN

**We know what a PLC smells like when it's burning.**

MATESTAIN is an Argentine industrial automation and software company. We program PLCs, commission production lines, and build software for the people who run them.
Everything we ship is open source unless a client contract says otherwise.

---

## What we do

**Field Engineering**
PLC programming (Siemens S7, Rockwell), robotics, end-of-line automation, site commissioning, remote support, operator training.
Industries: food & beverage, dairy, packaging, steel.

**Software Products**
Scalable tools for industrial environments — built by engineers who've spent time on the plant floor, not just in front of a screen.

| Product                                             | Description                                    |
| --------------------------------------------------- | ---------------------------------------------- |
| [Plantwise](https://github.com/matestain/plantwise) | Unified industrial plant intelligence platform |
